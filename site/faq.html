FAQ
Most common setup and usage questions.
How should files be named?
Use Steam App IDs, such as 220200 or 220200.zip.
Can I search by game name?
Yes. In drive mode, names are resolved through your game mapping.
Does local demo mode support real Drive downloads?
No. Demo mode uses local placeholder downloads. Use the Railway Node server for real API access.
